# 📊 Sample Superstore Data Analysis Using Python

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) on the Sample Superstore dataset using Python. The objective is to analyze sales performance, profitability, customer segments, product categories, and regional trends to generate actionable business insights.

Using data analysis and visualization techniques, the project identifies key factors affecting sales and profit, helping businesses make data-driven decisions. The Sample Superstore dataset is widely used for learning and practicing data analytics concepts.

---

## 🎯 Project Objectives

* Analyze sales and profit performance.
* Identify top-performing products and categories.
* Discover loss-making segments and regions.
* Understand customer and regional trends.
* Create meaningful visualizations for business insights.
* Support data-driven decision-making.

---

## 🔍 Exploratory Data Analysis

The analysis includes:

### Data Cleaning

* Handling missing values
* Removing duplicates
* Data type conversion
* Data quality checks

### Sales Analysis

* Total Sales Performance
* Sales by Category
* Sales by Sub-Category
* Sales by Region

### Profit Analysis

* Profit by Category
* Profit by Region
* Loss-Making Products
* Discount Impact on Profit

### Customer Analysis

* Customer Segments
* Purchasing Trends
* Regional Customer Behavior

### Visualization

* Bar Charts
* Line Charts
* Pie Charts
* Heatmaps
* Correlation Analysis

---

## 📈 Key Insights

* Certain categories contribute significantly to total sales.
* Some products generate high sales but low profits.
* Regional performance varies considerably.
* Discounts can negatively impact profitability.
* Customer segments exhibit different purchasing behaviors.
